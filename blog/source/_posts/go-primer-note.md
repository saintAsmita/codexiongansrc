# Go基础笔记摘要

本文只是介绍在基础阶段，Go语言中一些需要留意的特性。

1. 无符号整数一般用于特殊的位处理，二进制文件处理等，一般用的不多；
2. 浮点数优先选择float64, 32的误差可能比较大；
3. NaN 之间用 == ，总是不成立；
<!--more-->
4. 考虑到编码，字符串的第i个字节，不一定是第i个字符。[i]取的是第i个字符；
5. 子串操作共用底层数据，因此开销低廉；
6. 字符串和[]byte之间可以互相转换，会有拷贝行为；
7. 无类型int转的时候，大小不确定，而float那些都是确定的；
